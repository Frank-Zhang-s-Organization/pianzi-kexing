# Code of Conduct / 协作准则

> 中文优先；英文讨论同样欢迎。  
> Chinese-first, English discussion is also welcome.

## 我们希望这里是什么样的协作环境

这个项目欢迎：

- 技术上严谨的讨论
- 对高风险系统的审慎质疑
- 对误报、审计、边界和合规的认真补充

这个项目不欢迎：

- 煽动性、羞辱性或人身攻击式表达
- 鼓励绕过授权、绕过合规、绕过人工复核的建议
- 把项目导向“自动定罪”或“公共空间实时监控”的推动

## 讨论原则

- 讨论问题，不攻击个人
- 先给证据，再下判断
- 对高风险方向保持更高的论证门槛

## 执行

如果讨论或 PR 明显违背项目边界，维护者可以关闭 issue、拒绝 PR 或限制互动。
