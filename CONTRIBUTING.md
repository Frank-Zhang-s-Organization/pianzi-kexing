# CONTRIBUTING / 贡献指南

> 中文优先；英文 PR、issue 和文档补充同样欢迎。  
> Chinese-first; English PRs, issues, and documentation contributions are welcome.

欢迎贡献，但请先理解这个项目的前提：

- 它是合规优先的研究骨架
- 它不是实时监控产品
- 功能提议必须先过边界审查

## 提交贡献前请先检查

1. 是否符合 `docs/COMPLIANCE_BOUNDARY.md`
2. 是否符合 `AGENTS.md`
3. 是否会引入真实生物特征模板或高风险数据源
4. 是否会削弱人工复核、审计或撤销机制

## 更容易被接受的贡献

- 文档完善
- schema 设计
- policy / audit 模块
- 示例清单
- 测试补充

## 更可能被拒绝的贡献

- 真实识别引擎
- 民间风险名单采集器
- 面向公共空间的提示逻辑
- 会诱导项目越界的 UI / 文案
